# Еженедельные отчёты и обратная связь

Каждое воскресенье система автоматически собирает еженедельный отчёт по клиенту. Клиент подтверждает отправку, а ты оставляешь структурированную обратную связь.

---

## Где находится

Карточка клиента → вкладка **"Отчёты"** (`/curator/clients/:id?tab=reports`).

---

## Список отчётов

Отчёты отображаются списком с указанием:

- Номер недели и период (дата начала -- дата окончания)
- Дата отправки
- Статус: **"Ожидает обратной связи"** или **"Обратная связь дана"**

Нажми на отчёт, чтобы развернуть детали.

---

## Содержимое отчёта

Отчёт автоматически собирает данные за неделю:

- Средние калории, белки, жиры, углеводы
- Средний вес и изменение за неделю
- Количество шагов
- Количество тренировок
- Наличие еженедельного фото
- Прогресс по задачам

---

## Обратная связь

Нажми **"Оставить обратную связь"** на отчёте без фидбека. Откроется форма.

### Структура обратной связи

| Категория | Оценка | Комментарий |
|-----------|--------|-------------|
| **Питание** | Отлично / Хорошо / Нужно улучшить | Текстовый комментарий |
| **Активность** | Отлично / Хорошо / Нужно улучшить | Текстовый комментарий |
| **Вода** | Отлично / Хорошо / Нужно улучшить | Текстовый комментарий |
| **Фото** | Загружено / Не загружено | -- |

Плюс два текстовых поля:

| Поле | Описание | Обязательно |
|------|----------|-------------|
| **Итог** | Общий комментарий по неделе | Да |
| **Рекомендации** | Что делать на следующей неделе | Нет |

### Цветовое кодирование оценок

| Оценка | Цвет | Когда ставить |
|--------|------|---------------|
| **Отлично** | Зелёный | Клиент выполнил план на 90%+ |
| **Хорошо** | Жёлтый | Выполнение 70-90%, есть мелкие отклонения |
| **Нужно улучшить** | Красный | Выполнение <70%, систематические нарушения |

---

## Что видит клиент

После отправки обратной связи клиент получает уведомление и видит в дашборде:

- Карточку с оценками по категориям (цветовые индикаторы)
- Итоговый комментарий
- Рекомендации на следующую неделю
- Карточка видна до конца следующей недели

---

## Рекомендации по обратной связи

- **Давай фидбек в течение 24 часов** после получения отчёта -- клиент ждёт
- **Начинай с позитива** -- отметь, что получилось хорошо
- **Будь конкретен** в рекомендациях: "Увеличь белок до 120г" вместо "Ешь больше белка"
- **Привязывай к задачам** -- если ставишь рекомендацию, создай соответствующую задачу
- **Используй все категории** -- клиенту важна оценка по каждому направлению
