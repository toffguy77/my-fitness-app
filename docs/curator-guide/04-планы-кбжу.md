# Планы КБЖУ

План КБЖУ -- это недельные целевые показатели по калориям, белкам, жирам и углеводам, которые ты назначаешь клиенту. Клиент видит план в своём дашборде и ориентируется на него при заполнении дневника.

---

## Где находится

Карточка клиента → вкладка **"План"** (`/curator/clients/:id?tab=plan`).

---

## Текущий план

Если у клиента есть активный план, ты увидишь:

- **Целевые значения:** калории, белки, жиры, углеводы
- **Период:** дата начала -- дата окончания
- **Комментарий** (если был указан)
- Кнопка **"Скорректировать"** -- редактирование текущего плана

Если активного плана нет -- отображается пустое состояние с приглашением создать план.

---

## Создание плана

Нажми кнопку **"+"** (круглая кнопка в правом нижнем углу экрана).

Откроется форма:

| Поле | Описание | Обязательно |
|------|----------|-------------|
| Калории | Целевая калорийность в сутки | Да |
| Белки (г) | Граммы белка в сутки | Да |
| Жиры (г) | Граммы жиров в сутки | Да |
| Углеводы (г) | Граммы углеводов в сутки | Да |
| Дата начала | По умолчанию -- понедельник текущей недели | Да |
| Дата окончания | По умолчанию -- воскресенье текущей недели | Да |
| Комментарий | Пояснение для клиента | Нет |

### Рекомендации по заполнению

- Ориентируйся на данные клиента: возраст, пол, вес, рост, уровень активности, цель
- Для снижения веса -- дефицит 300-500 ккал от нормы
- Для набора -- профицит 200-300 ккал
- Баланс макросов: белки ~1.5-2г/кг, жиры ~0.8-1г/кг, остальное -- углеводы
- Используй комментарий, чтобы объяснить логику ("Увеличил белок, т.к. добавили силовые")

---

## Корректировка плана

Нажми **"Скорректировать"** на текущем плане. Откроется та же форма с предзаполненными значениями. Даты периода при редактировании не меняются.

Корректируй план, когда:
- Клиент не справляется с текущими целями (систематически недобирает или перебирает)
- Изменился вес клиента -- пересчитай калорийность
- Изменилась активность или цель
- Прошла неделя -- пора задать новый план

---

## Новый план при наличии активного

Если текущий план ещё действует, но нужно создать новый (например, на следующую неделю) -- нажми **"+"**. Новый план станет активным, предыдущий уйдёт в историю.

---

## История планов

Под текущим планом -- раскрываемый блок **"История планов"** с числом прошлых планов. Каждый элемент содержит:

- Период (даты начала и окончания)
- Значения КБЖУ
- Комментарий (если был)

История помогает отследить, как менялись целевые показатели со временем.

---

## Что видит клиент

В дашборде клиента отображается текущий активный план:
- Целевые значения КБЖУ
- Комментарий куратора
- Прогресс-бары относительно плана по каждому макронутриенту
