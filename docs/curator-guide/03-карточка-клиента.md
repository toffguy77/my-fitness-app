# Карточка клиента

Карточка клиента (`/curator/clients/:id`) -- основной экран работы с конкретным клиентом. Здесь собрана вся информация: питание, вес, вода, задачи, планы, отчёты.

---

## Шапка карточки

Вверху карточки:

- **Имя и аватар** клиента
- Кнопка **"Написать"** -- быстрый переход в чат
- **"Подробнее"** -- раскрывает дополнительную информацию:
  - Email, Telegram, Instagram
  - Возраст, пол, рост
  - Уровень активности, фитнес-цель
  - Часовой пояс

---

## Вкладки

Под шапкой -- четыре вкладки. Переключение через query-параметр `?tab=`.

| Вкладка | Содержимое |
|---------|-----------|
| **Обзор** | Дневные записи: КБЖУ, вес, вода, шаги, тренировки, фото |
| **План** | Текущий план КБЖУ + история планов |
| **Задачи** | Список задач + создание новых |
| **Отчёты** | Еженедельные отчёты + обратная связь |

---

## Вкладка "Обзор"

Показывает данные клиента за последние 7 дней (настраивается до 30).

### Что видно по каждому дню

- **КБЖУ** -- калории, белки, жиры, углеводы с прогресс-барами относительно плана
- **Записи о еде** -- каждый приём пищи с названием, весом порции и КБЖУ
- **Вес** -- если клиент взвешивался в этот день
- **Вода** -- стаканы из целевого количества
- **Шаги** -- если клиент логирует шаги
- **Тренировка** -- тип и продолжительность

### График веса

Отдельный блок с историей веса:
- Все взвешивания за период
- Тренд: вверх / вниз / стабильно
- Целевой вес (если задан)

### Целевой вес и норма воды

Ты можешь задать клиенту:
- **Целевой вес** -- отображается на графике веса как целевая линия
- **Норму воды** -- количество стаканов в день (от 1 до 30)

### Фото

Еженедельные фото прогресса, загруженные клиентом.

---

## Навигация

- **Стрелка назад** -- возврат к списку клиентов / хабу
- **"Написать"** -- переход в чат с клиентом
- **Вкладки** -- переключение между разделами
