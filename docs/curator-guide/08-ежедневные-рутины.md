# Ежедневные и еженедельные рутины

Эффективная работа куратора строится на регулярных рутинах. Здесь описан рекомендуемый распорядок.

---

## Ежедневная рутина

### Утро (начало рабочего дня)

1. **Открой хаб** → посмотри метрики-сводку
2. **Проверь список "Требуют внимания"** → начни с приоритета 1 (красные алерты КБЖУ)
3. **Ответь на непрочитанные сообщения** → клиенты ждут ответов
4. **Проверь просроченные задачи** → напиши клиентам, которые пропустили дедлайн

### В течение дня

5. **Отслеживай входящие сообщения** → бейдж на иконке чата показывает непрочитанные
6. **Создавай записи о еде** → когда клиенты присылают фото в чат
7. **Реагируй на алерты** → система обновляет приоритеты в реальном времени

### Вечер

8. **Финальная проверка хаба** → убедись, что все критические вопросы решены
9. **Проверь клиентов без активности** → если кто-то не заполнял дневник 2+ дня, напиши в чат

---

## Еженедельная рутина

### Понедельник -- планирование

1. **Обнови планы КБЖУ** для клиентов, у которых план истёк или нуждается в корректировке
2. **Поставь задачи на неделю** -- 2-3 задачи для каждого клиента
3. **Проверь динамику** в блоке бенчмарков -- как твои показатели за прошлую неделю

### Среда -- промежуточная проверка

4. **Пройди по всем клиентам** → вкладка "Обзор" → оцени, как идёт неделя
5. **Скорректируй планы** если видишь, что клиент систематически не попадает в цели
6. **Проверь прогресс задач** → кто выполняет, кто нет

### Воскресенье -- обратная связь

7. **Дай обратную связь по еженедельным отчётам** → хаб показывает "Ожидают обратной связи: N"
8. **Пройди по каждому отчёту:**
   - Оцени питание, активность, воду
   - Отметь наличие фото
   - Напиши итог и рекомендации
9. **Подготовь планы на следующую неделю** если нужно

---

## Чек-лист ежедневных действий

| Действие | Когда | Приоритет |
|----------|-------|-----------|
| Проверить хаб и алерты | Утро | Высокий |
| Ответить на сообщения | В течение дня | Высокий |
| Создать записи о еде по фото | При поступлении | Средний |
| Проверить клиентов без активности | Вечер | Средний |
| Обновить планы (понедельник) | Утро | Высокий |
| Дать обратную связь (воскресенье) | До конца дня | Высокий |

---

## Приоритизация

Если времени мало, фокусируйся на:

1. **Красные алерты** -- клиент серьёзно отклонился от плана
2. **Непрочитанные сообщения** -- клиент ждёт ответа
3. **Просроченные задачи** -- клиент может потерять мотивацию
4. **Обратная связь по отчётам** -- клиент ожидает фидбек

Клиенты на зелёном уровне (выполняют план, нет алертов) требуют минимум внимания -- достаточно проверить раз в 2-3 дня.
