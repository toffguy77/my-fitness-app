# Индекс диаграмм My Fitness App

**Версия документа:** 1.0  
**Дата создания:** Январь 2025  
**Статус:** Актуальная реализация v4.0+

## Обзор

Этот документ является индексом всех диаграмм My Fitness App, организованных по типам и назначению.

## Типы диаграмм

### 1. Архитектурные диаграммы
Диаграммы, описывающие техническую архитектуру системы, компоненты и развертывание.

**Документ:** [Diagrams_Architecture.md](./Diagrams_Architecture.md)

**Содержание:**
- Общая архитектура системы
- Компонентная архитектура
- Диаграмма развертывания
- Схема взаимодействия Frontend-Backend-Database

---

### 2. Диаграммы потоков данных
Диаграммы, описывающие потоки данных для ключевых процессов.

**Документ:** [Diagrams_Data_Flow.md](./Diagrams_Data_Flow.md)

**Содержание:**
- Поток данных при регистрации
- Поток данных при вводе питания
- Поток данных в чате
- Поток данных при OCR обработке
- Поток данных при автоматической деактивации подписок
- Поток данных при поиске продуктов

---

### 3. Диаграммы пользовательских потоков
Диаграммы, описывающие полные user journey для всех ролей.

**Документ:** [Diagrams_User_Flows.md](./Diagrams_User_Flows.md)

**Содержание:**
- Полный user journey для клиента (Free)
- Полный user journey для клиента (Premium)
- Полный user journey для тренера
- Onboarding flow
- Subscription lifecycle flow

---

### 4. Диаграммы состояний
Диаграммы, описывающие состояния различных сущностей системы.

**Документ:** [Diagrams_State_Machine.md](./Diagrams_State_Machine.md)

**Содержание:**
- Состояния подписки
- Состояния дня (is_completed)
- Состояния онбординга
- Состояния сообщений в чате
- Состояния OCR обработки
- Состояния достижений

---

### 5. Sequence диаграммы
Диаграммы последовательности для ключевых взаимодействий.

**Документ:** [Diagrams_Sequence.md](./Diagrams_Sequence.md)

**Содержание:**
- Регистрация и онбординг
- Ввод питания и сохранение
- Чат между тренером и клиентом
- OCR обработка фото
- Автоматическая деактивация подписок
- Сохранение заметки тренера
- Использование инвайт-кода

---

### 6. ER-диаграмма базы данных
Диаграмма сущность-связь для базы данных.

**Документ:** [Diagrams_Database_ER.md](./Diagrams_Database_ER.md)

**Содержание:**
- Полная ER-диаграмма всех таблиц
- Связи между таблицами
- Индексы
- Ограничения (Constraints)
