# BURCEV — Документация

BURCEV — платформа для трекинга питания и фитнеса с персональным куратором. Приложение помогает вести дневник питания, отслеживать прогресс, общаться с тренером и получать образовательный контент по здоровому образу жизни.

## Быстрый старт (клиент)

1. Зарегистрируйся с email и паролем
2. Подтверди email 6-значным кодом
3. Пройди 5-шаговый онбординг (фото, настройки, параметры тела, соцсети, Apple Health)
4. Начни вести дневник питания и общайся с назначенным куратором

## Быстрый старт (куратор)

1. Открой хаб — посмотри метрики и кому нужно внимание
2. Ответь на непрочитанные сообщения
3. Проверь красные алерты КБЖУ и просроченные задачи
4. В понедельник — обнови планы и поставь задачи на неделю
5. В воскресенье — дай обратную связь по еженедельным отчётам
