# Настройка профиля

В настройках профиля ты можешь указать свои физические параметры, цели, предпочтения по локализации и привязать социальные сети. Все эти данные используются для персонализации расчётов КБЖУ и работы приложения.

---

## Как перейти в настройки

1. Нажми на свой аватар или имя в шапке приложения.
2. Перейди в раздел **Профиль**.
3. Выбери нужный подраздел настроек.

---

## Разделы настроек

### Профиль и локализация

Основные данные и региональные предпочтения:

| Параметр              | Описание                                       | Значения                              |
|-----------------------|------------------------------------------------|---------------------------------------|
| **Аватар**            | Фото профиля                                   | JPEG, PNG, WebP; макс. 5 МБ          |
| **Имя**               | Отображаемое имя                               | Произвольный текст                    |
| **Рост**              | Рост в сантиметрах                             | 50--300 см                            |
| **Язык**              | Язык интерфейса                                | Русский / Английский                  |
| **Единицы измерения** | Система мер                                    | Метрические / Имперские               |
| **Часовой пояс**      | Часовой пояс для корректного отображения дат   | Любой валидный часовой пояс (напр. Europe/Moscow) |

### Тело и цели

Физические параметры и фитнес-цель:

| Параметр              | Описание                                       | Значения                              |
|-----------------------|------------------------------------------------|---------------------------------------|
| **Дата рождения**     | Для расчёта базового метаболизма               | Формат ГГГГ-ММ-ДД                    |
| **Биологический пол** | Для точного расчёта BMR                        | Мужской / Женский                     |
| **Рост**              | Рост в сантиметрах                             | 50--300 см                            |
| **Текущий вес**       | Вводится на дашборде (не в настройках)         | --                                    |
| **Целевой вес**       | Желаемый вес                                   | 20--500 кг                            |
| **Уровень активности**| Количество тренировок в неделю                 | См. таблицу ниже                      |
| **Цель**              | Фитнес-цель                                   | Снижение веса / Поддержание / Набор массы |

**Уровни активности:**

| Уровень              | Описание                          |
|----------------------|-----------------------------------|
| Сидячий образ жизни  | Офис, минимум движения            |
| Лёгкая активность    | 1--2 тренировки в неделю          |
| Умеренная активность | 3--5 тренировок в неделю          |
| Высокая активность   | 6--7 тренировок в неделю          |

> :warning: Важно: При сохранении параметров тела (рост, дата рождения, пол, уровень активности, цель) автоматически запускается пересчёт твоих целевых КБЖУ. Ты увидишь уведомление "Нормы пересчитаны", когда пересчёт завершится.

### Социальные сети

Привяжи свои аккаунты для связи с куратором:

| Сеть        | Что указать                          |
|-------------|--------------------------------------|
| **Telegram** | Имя пользователя (без @)            |
| **Instagram**| Имя пользователя (без @)            |

При сохранении система проверяет существование указанных аккаунтов. Если аккаунт не найден, ты получишь сообщение об ошибке.

### Apple Health

- **Синхронизация с Apple Health** -- переключатель для включения/отключения интеграции
- Доступно только на устройствах iOS

### Уведомления

Настройка уведомлений о контенте (подробнее в разделе [Уведомления](./06-уведомления.md)):

- Режим "Не беспокоить" -- отключает все контентные уведомления
- Управление по категориям -- отключай ненужные темы

## Пошаговая инструкция: загрузка аватара

1. Перейди в **Профиль > Настройки профиля**.
2. Нажми на область аватара.
3. Выбери изображение с устройства.
4. Аватар загрузится автоматически.

Чтобы удалить аватар, нажми кнопку удаления рядом с фото.

**Требования к файлу:**
- Формат: JPEG, PNG или WebP
- Размер: не более 5 МБ

---

## Пошаговая инструкция: настройка параметров тела

1. Перейди в **Профиль > Тело и цели**.
2. Заполни поля: дата рождения, биологический пол, рост, целевой вес.
3. Выбери уровень активности из выпадающего списка.
4. Выбери фитнес-цель: снижение веса, поддержание или набор массы.
5. Нажми **Сохранить**.
6. Дождись уведомления "Нормы пересчитаны" -- это подтверждение, что КБЖУ обновлены.

> :warning: Важно: Текущий вес вводится не в настройках, а на дашборде. Перейди на дашборд, чтобы записать актуальный вес.

---

## Возможности

- Мгновенное сохранение настроек локализации (язык, единицы, часовой пояс) -- без кнопки "Сохранить"
- Автоматический пересчёт КБЖУ при изменении параметров тела
- Валидация социальных аккаунтов при сохранении
- Удаление аккаунта (через кнопку в настройках профиля)

---

## Ограничения

- Аватар: максимум 5 МБ, только изображения (JPEG, PNG, WebP)
- Рост: от 50 до 300 см
- Целевой вес: от 20 до 500 кг
- Биологический пол: только мужской или женский (используется для формулы расчёта BMR)
- Удаление аккаунта пока в разработке -- свяжись с администратором для удаления
- Нельзя изменить email из настроек

---

## Советы

- Заполни все параметры тела для максимально точного расчёта КБЖУ
- Если ты изменил уровень активности (например, стал тренироваться чаще), обнови настройки -- КБЖУ пересчитаются автоматически
- Укажи Telegram -- так куратору будет проще связаться с тобой вне платформы
- При смене часового пояса (поездка, переезд) обнови настройки, чтобы данные по питанию корректно привязывались к дням
