# Общение с куратором

Чат с куратором -- это твой прямой канал связи с персональным наставником. Здесь ты можешь задавать вопросы о питании, тренировках, обсуждать трудности и получать рекомендации. Чат работает в реальном времени через WebSocket-соединение.

---

## Пошаговые инструкции

### 1. Назначение куратора

При регистрации система автоматически назначает тебе куратора. Выбирается наименее загруженный из доступных кураторов. Тебе не нужно ничего делать -- чат с куратором появится автоматически.

Если у тебя уже есть активная связь с куратором, но чата еще нет, система создаст его при первом входе.

### 2. Открытие чата

1. Нажми на иконку чата в нижней навигации дашборда
2. Откроется список твоих чатов (обычно один -- с твоим куратором)
3. Нажми на имя куратора, чтобы открыть переписку

Если у тебя есть непрочитанные сообщения, на иконке чата в нижней навигации будет отображаться бейдж с количеством.

### 3. Отправка текстового сообщения

1. Введи текст в поле "Сообщение..." внизу экрана
2. Нажми кнопку отправки (стрелка вверх) или клавишу Enter
3. Твое сообщение появится справа с синим фоном
4. Время отправки отображается под каждым сообщением

### 4. Отправка файла или изображения

1. Нажми на иконку скрепки слева от поля ввода
2. Выбери файл с устройства
3. Файл отобразится над полем ввода (имя файла с возможностью отмены)
4. Нажми кнопку отправки для загрузки файла

Поддерживаемые типы сообщений:

| Тип | Как выглядит |
|---|---|
| Текст | Текстовый пузырь (синий -- твои, серый -- куратора) |
| Изображение | Превью изображения с возможностью открыть в полном размере |
| Файл | Блок с именем файла и ссылкой для скачивания |
| Запись о еде | Карточка с названием продукта и показателями КБЖУ |

> **Важно:** Максимальный размер загружаемого файла -- **10 МБ**.

### 5. Индикатор набора текста

Когда куратор печатает сообщение, ты увидишь анимированный индикатор "печатает..." под последним сообщением. Куратор тоже видит, когда ты набираешь текст.

Индикатор отправляется с задержкой (debounce 2 секунды) -- он срабатывает при первом нажатии клавиши, а затем ждет паузы.

### 6. Статус прочтения сообщений

Система отслеживает прочтение сообщений:

1. Когда ты открываешь чат, все входящие сообщения автоматически отмечаются как прочитанные
2. Счетчик непрочитанных обнуляется для этого чата
3. Общий счетчик непрочитанных обновляется в нижней навигации

Обновление счетчика происходит в реальном времени -- куратор получает обновленный счетчик сразу после отправки каждого нового сообщения.

### 7. Непрочитанные сообщения

Количество непрочитанных сообщений отображается:
- На иконке чата в нижней навигации (общий бейдж)
- В списке чатов рядом с именем куратора (счетчик по каждому чату)

Счетчик обновляется в реальном времени через WebSocket -- ты получаешь уведомление моментально, как только куратор отправит сообщение.

### 8. Создание записи о еде из чата

Куратор может создать запись о еде прямо из чата:

1. Куратор отправляет сообщение типа "food_entry" с данными о продукте
2. В чате появляется карточка с информацией:
   - Название продукта
   - Тип приема пищи (завтрак, обед, ужин, перекус)
   - Вес порции
   - Калории, белки, жиры, углеводы
3. Запись автоматически добавляется в твой дневник питания за текущий день

> **Важно:** Записи о еде из чата может создавать только куратор. Это удобно, когда ты отправил фото еды, а куратор рассчитал и записал КБЖУ за тебя.

### 9. Автоподгрузка старых сообщений

При прокрутке чата вверх старые сообщения загружаются автоматически:

1. Открой чат -- загрузятся последние 50 сообщений
2. Если есть более ранние сообщения, вверху появится кнопка "Загрузить ещё"
3. Нажми на нее для загрузки следующей порции (до 100 сообщений за раз)

Сообщения группируются по дням с разделителями дат (например, "7 марта 2026").

### 10. Еженедельный отчет

Еженедельный отчет отправляется через дашборд (не через чат):

1. В воскресенье на дашборде появится кнопка "Отправить недельный отчет"
2. Перед отправкой убедись, что за неделю заполнены данные по питанию и весу минимум за 5 дней, а также загружено фото
3. Отчет автоматически собирает итоги: средние калории, средний вес, шаги, количество тренировок
4. Куратор получает уведомление и может оставить обратную связь

### 11. Задачи от куратора

Задачи отображаются в разделе "Задачи" на дашборде:

1. Куратор назначает задачи с заголовком, описанием и дедлайном
2. Каждая задача привязана к определенной неделе
3. Отметь задачу выполненной, нажав на кружок слева от названия
4. Задачи со статусом "просрочена" подсвечиваются красным

Подробнее о задачах -- в разделе [Отслеживание прогресса](./03-отслеживание-прогресса.md).

## Возможности

- Обмен сообщениями в реальном времени через WebSocket
- Отправка текстовых сообщений, изображений и файлов
- Индикатор набора текста (видно, когда куратор печатает)
- Автоматическая отметка прочитанных сообщений при открытии чата
- Бейджи непрочитанных сообщений в навигации
- Создание записей о еде куратором прямо из чата
- Пагинация сообщений при прокрутке (курсорная навигация)
- Группировка сообщений по дням с датами-разделителями
- Автоматическое подключение к WebSocket с реконнектом при разрыве соединения
- Автоматическое назначение куратора при регистрации

## Ограничения

- Максимальный размер загружаемого файла -- 10 МБ
- Одновременно загружается до 50 сообщений (далее -- по кнопке "Загрузить ещё"), максимум 100 за один запрос
- Записи о еде из чата может создавать только куратор, не клиент
- Кнопка "Ввести КБЖУ" на фото в чате доступна только куратору (отображается на изображениях от клиента)
- Выбор куратора происходит автоматически -- самостоятельно сменить куратора нельзя
- При потере интернета WebSocket-соединение разрывается; переподключение происходит автоматически с нарастающей задержкой (от 1 до 30 секунд)
- Нельзя удалять или редактировать уже отправленные сообщения

## Советы

- **Регулярно отправляй фото прогресса** -- это помогает куратору оценить результаты визуально и скорректировать план
- **Задавай вопросы куратору** -- не стесняйся спрашивать о правильности подсчета КБЖУ, технике упражнений или подборе продуктов
- **Сообщай о трудностях** -- если тебе сложно соблюдать план питания или ты чувствуешь усталость, сообщи куратору. Он скорректирует программу под твои текущие возможности
- **Фотографируй еду перед приемом пищи** -- отправь фото в чат, и куратор сможет прямо из чата создать запись с точными данными КБЖУ
- **Проверяй непрочитанные сообщения** -- обращай внимание на бейдж в нижней навигации, чтобы не пропустить важные рекомендации от куратора
- **Отправляй еженедельный отчет вовремя** -- в воскресенье не забудь нажать кнопку на дашборде, чтобы куратор получил сводку и смог спланировать следующую неделю
- **Выполняй задачи от куратора** -- отмечай задачи выполненными сразу после завершения, чтобы куратор видел твой прогресс
- **Не игнорируй просроченные задачи** -- если ты не успеваешь, напиши куратору, чтобы он скорректировал дедлайны
