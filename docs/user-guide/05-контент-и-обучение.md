# Контент и обучение

Раздел **Контент** -- это лента полезных статей от твоих кураторов и экспертов платформы. Здесь ты найдёшь образовательные материалы по питанию, тренировкам, рецептам и мотивации. Статьи пишут кураторы специально для своих подопечных, а часть контента доступна всем пользователям.

---

## Как пользоваться лентой

1. Перейди в раздел **Контент** в главном меню приложения.
2. По умолчанию отображаются все доступные тебе статьи, отсортированные по дате публикации (сначала новые).
3. Используй **фильтр по категориям** в верхней части экрана, чтобы показать статьи только нужной темы.
4. Нажми на карточку статьи, чтобы прочитать её целиком.
5. Если статей много, нажми кнопку **Загрузить ещё** внизу ленты.

---

## Категории контента

| Категория     | Что внутри                                      |
|---------------|------------------------------------------------|
| Питание       | Основы нутрициологии, советы по рациону         |
| Тренировки    | Программы, техника упражнений, планирование     |
| Рецепты       | Полезные рецепты с расчётом КБЖУ                |
| Здоровье      | Восстановление, сон, профилактика               |
| Мотивация     | Истории успеха, психология привычек             |
| Общее         | Новости платформы и другие полезные материалы    |

Чтобы увидеть все категории, нажми кнопку **Все** в фильтре.

---

## Возможности

- **Фильтрация по категориям** -- выбери одну категорию или просматривай всё сразу
- **Полнотекстовые статьи** -- с поддержкой изображений, таблиц, списков и форматированного текста (Markdown)
- **Обложки статей** -- карточки в ленте отображают обложку, заголовок, краткое описание, автора и дату публикации
- **Уведомления о новых статьях** -- при публикации новой статьи ты получишь уведомление (если не отключил их в настройках)
- **Публичный доступ** -- статьи с аудиторией "для всех" доступны без авторизации по прямой ссылке

---

## Какие статьи ты видишь

Видимость статей зависит от настроек куратора:

- **Для всех** -- статья доступна всем пользователям платформы, включая незарегистрированных
- **Для моих клиентов** -- статья видна только подопечным данного куратора
- **Для выбранных** -- статья видна только конкретным пользователям, которых куратор указал вручную

> :warning: Важно: Если ты не видишь статью, к которой тебе дали ссылку, возможно, она предназначена для другой аудитории. Обратись к своему куратору.

---

## Ограничения

- Ты не можешь создавать, редактировать или удалять статьи -- это функция куратора
- Лента загружается по 20 статей за раз
- Для чтения персонализированного контента (статьи для клиентов куратора) необходимо быть авторизованным
- Комментарии к статьям пока не поддерживаются -- используй чат с куратором для обсуждения

---

## Советы

- Проверяй ленту регулярно -- кураторы могут планировать публикации на определённое время
- Если тебе приходит слишком много уведомлений о новых статьях, ты можешь отключить их по категориям в **Настройки > Уведомления**
- Делись полезными статьями с друзьями -- статьи с публичным доступом можно открыть по прямой ссылке без регистрации
