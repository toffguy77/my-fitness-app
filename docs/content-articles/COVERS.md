## Как скачать

### Вариант 1: Прямая ссылка (рекомендуется)
Используй URL с параметром `?w=800&q=80` — он вернёт оптимизированное изображение шириной 800px.

```bash
# Пример скачивания через curl
curl -L "https://images.unsplash.com/photo-1498837167922-ddd27525d352?w=800&q=80" -o cover-01.jpg
```

### Вариант 2: Страница на Unsplash
Перейди по ссылке на Unsplash, нажми "Download" и выбери нужный размер.

### Рекомендуемые параметры
- **Ширина:** 800px (`w=800`) — достаточно для карточки статьи
- **Качество:** 80% (`q=80`) — баланс между качеством и размером файла
- **Формат:** JPEG (по умолчанию) или добавь `&fm=webp` для WebP
