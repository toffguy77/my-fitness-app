## 11. Офлайн-Режим

### Функционал

```
Работает офлайн:
✓ Просмотр истории (кэш 7 дней)
✓ Добавление записей (синхронизация позже)
✓ Просмотр контента (кэшированного)
✓ Выполнение тренировок
✓ Отметка привычек

Не работает офлайн:
✗ Чат с куратором (реал-тайм)
✗ Загрузка нового контента
✗ Поиск продуктов (требует сервер)

При восстановлении связи:
→ Автоматическая синхронизация всех данных
→ Отправка сообщений из очереди
```
