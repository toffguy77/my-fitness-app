# Трекер Тренировок (Workout Tracker)

## Обзор

Модуль для логирования тренировок с детальной информацией о типе, длительности, нагрузке, самочувствии и работоспособности.

---

## Экраны в Разделе

1. **Workout Type Selection** (Выбор типа тренировки)
2. **Workout Time Input** (Ввод длительности)
3. **Workout Load Input** (Оценка нагрузки)
4. **Workout Feel Input** (Самочувствие)
5. **Workout Work Capacity** (Работоспособность)
6. **Workout Done** (Подтверждение завершения)

---

### Опции Ввода

#### Простой Режим (Slider)
- Диапазон: 0-180 минут
- Шаг: 5 минут
- Быстрые кнопки для популярных длительностей

#### Точный Режим (Time Picker)
- Выбор времени начала
- Выбор времени окончания
- Автоматический расчет длительности
- Полезно для длинных тренировок (>3 часов)

### Валидация

#### Предупреждения
```javascript
if (duration < 10) {
  warning("Очень короткая тренировка. Вы уверены?");
}

if (duration > 180) {
  warning("Длинная тренировка! Не забывайте про восстановление.");
}
```

- **→ Workout Load**: После ввода длительности

### Метрики
- Средняя длительность тренировок по типам
- Распределение длительности
- % использования slider vs time picker

### Описание Уровней

```

RPE 3-4:  Легко (можете разговаривать свободно)
RPE 5-6:  Средне (умеренная нагрузка, небольшая усталость)
RPE 7-8:  Тяжело (интенсивная работа, трудно говорить)
RPE 9-10: Максимум (работа на пределе возможностей)
```

### Градации Самочувствия

```

  - Много энергии
  - Хорошее настроение
  - Готов тренироваться еще

😊 Хорошо (4/5)
  - Нормальная энергия
  - Небольшая усталость
  - Позитивные ощущения

  - Умеренная усталость
  - Нейтральное состояние
  - Завершил тренировку с трудом

😕 Не очень (2/5)
  - Сильная усталость
  - Нехватка сил
  - Пришлось снизить интенсивность

😫 Плохо (1/5)
  - Очень плохое самочувствие
  - Пришлось прекратить тренировку
  - Возможны негативные симптомы
```

### Анализ Данных

#### Алерты для Куратора
При выборе "Плохо" с симптомами:
```
Автоматическое уведомление куратору:

"⚠️ [Имя пользователя] отметил плохое
самочувствие на тренировке с симптомами:
- Головокружение
- Тошнота

Тип: Силовая, 60 мин, RPE: 9/10

[Связаться с клиентом]"
```

#### Паттерн-анализ
```
Система отслеживает:
- Частые жалобы → рекомендация обратиться к врачу
- Постоянно "плохо" → снижение нагрузок в плане
- Всегда "отлично" → возможность увеличения интенсивности
```

### Связи с Экранами

### Метрики

### Автоматический Расчет

#### Сожженные Калории
```javascript
function calculateCaloriesBurned(workout) {
  const MET = getMetValue(workout.type, workout.intensity);
  const weight = user.currentWeight; // в кг
  const duration = workout.duration; // в минутах

  // Формула: Калории = MET × вес (кг) × время (часы)
  const calories = MET * weight * (duration / 60);

  return Math.round(calories);
}

// MET значения для разных активностей
const MET_VALUES = {
  'Силовая (низкая)': 3.0,
  'Силовая (средняя)': 5.0,
  'Силовая (высокая)': 6.0,
  'Бег (6 мин/км)': 10.0,
  'Бег (5 мин/км)': 11.5,
  'Йога (Хатха)': 2.5,
  'Йога (Виньяса)': 4.0,
  // и т.д.
};
```

## Бизнес-Правила

### Частота Тренировок
- Рекомендация: 3-5 раз в неделю
- Обязательные дни отдыха
- Алерт при перетренированности (>6 дней подряд с RPE >7)

### Прогрессия Нагрузок
```javascript
// Автоматические рекомендации
if (lastWeek.avgRPE < 6 && thisWeek.avgRPE < 6) {
  suggest("Попробуйте увеличить нагрузку на следующей неделе");
}

if (tonnage.thisWeek > tonnage.lastWeek * 1.2) {
  warn("Осторожно! Резкое увеличение объема может привести к травмам");
}
```

### Consistency Rewards
```
Стрики (Streaks):
- 7 дней подряд → Бейдж "Неделя силы"
```
