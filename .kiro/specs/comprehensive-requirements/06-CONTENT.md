# Контент и Сообщество

## Обзор

Раздел **Контент** — это образовательная и мотивационная часть приложения, где пользователь получает доступ к:
- Персонализированной ленте контента от куратора
- Программам тренировок и питания
- Базе знаний (Wiki)
- Избранному контенту
- Социальным взаимодействиям (лайки, комментарии)

---

## Структура Раздела

### Основные Экраны

1. **Content Main Feed** — Главная лента контента
2. **Content Item Detail** — Детальный просмотр контента
3. **Programs** — Программы тренировок и питания
4. **Program Detail** — Детали программы
5. **Wiki** — База знаний
6. **Favorites** — Избранный контент
7. **Comments** — Комментарии к контенту

---

## 1. Content Main Feed

### Назначение
Персонализированная лента контента от куратора: статьи, видео, рецепты, мотивационные посты.

### UI Компоненты

#### Заголовок
- **Название раздела**: "Контент"
- **Табы фильтров**:
  - Все
  - Статьи
  - Видео
  - Рецепты
  - Тренировки
  - Мотивация
- **Иконка фильтра**: Дополнительные фильтры (по дате, популярности)
- **Иконка поиска**: Поиск по контенту

#### Карточки Контента
Каждая карточка содержит:

```
┌─────────────────────────────────────┐
│ [Превью изображение/видео]          │
│                                     │
│ [Тип контента бейдж]    [❤️ 234]   │
│                                     │
│ Заголовок контента                  │
│ Краткое описание (2-3 строки)...   │
│                                     │
│ 👤 Имя куратора    📅 5 дней назад  │
│ ⏱️ 5 мин чтения    💬 12 комментариев│
└─────────────────────────────────────┘
```

#### Типы Контента

1. **Статья** 📄
   - Превью: Обложка статьи
   - Метаданные: Время чтения, дата публикации
   - Бейдж: "Статья"

2. **Видео** 🎥
   - Превью: Thumbnail с иконкой play
   - Метаданные: Длительность видео
   - Бейдж: "Видео"

3. **Рецепт** 🍳
   - Превью: Фото блюда
   - Метаданные: Время приготовления, калорийность
   - Бейдж: "Рецепт"

4. **Тренировка** 💪
   - Превью: Демо-фото упражнения
   - Метаданные: Длительность, уровень сложности
   - Бейдж: "Тренировка"

5. **Мотивация** ✨
   - Превью: Мотивационное изображение
   - Метаданные: Категория мотивации
   - Бейдж: "Мотивация"

#### Взаимодействия
- **Лайк**: Иконка сердца (заполненное/незаполненное)
- **Комментарий**: Переход к комментариям
- **Сохранить**: Добавить в избранное (иконка закладки)
- **Поделиться**: Нативный share sheet

#### Пустое Состояние
```
"📚 Контент появится скоро!
Ваш куратор подготавливает персонализированные материалы."
```

### Поведение

#### Загрузка Ленты
```javascript
async function loadContentFeed(filters) {
  const response = await api.get('/content/feed', {
    params: {
      type: filters.type, // 'all', 'article', 'video', etc.
      sort: filters.sort, // 'recent', 'popular', 'trending'
      page: filters.page,
      limit: 10
    }
  });

  return response.data.items.map(item => ({
    id: item.id,
    type: item.content_type,
    title: item.title,
    description: item.excerpt,
    thumbnail: item.thumbnail_url,
    author: item.curator_name,
    publishedAt: item.published_at,
    readTime: item.estimated_read_time_minutes,
    likes: item.likes_count,
    comments: item.comments_count,
    isLiked: item.user_has_liked,
    isFavorite: item.user_has_favorited
  }));
}
```

#### Infinite Scroll
- Автозагрузка следующей страницы при достижении 80% скролла
- Индикатор загрузки в конце списка
- Кэширование загруженных страниц

#### Pull-to-Refresh
- Обновление ленты с сервера
- Показ новых постов с анимацией
- Индикация количества новых постов: "↓ 3 новых поста"

### Фильтры и Поиск

#### Фильтры

- Поиск по тегам и категориям

- Популярные запросы

### Действия

#### Лайк
```javascript
async function toggleLike(contentId) {
  const response = await api.post(`/content/${contentId}/like`);

  if (response.data.liked) {
    // Анимация сердца (масштабирование + цвет)
    showAnimation('heart-pop');
    updateLikesCount(+1);
  } else {
    updateLikesCount(-1);
  }

  // Отправка аналитики
  analytics.track('content_liked', {
    content_id: contentId,
    content_type: contentType,
    action: response.data.liked ? 'like' : 'unlike'
  });
}
```

#### Сохранить в Избранное
```javascript
async function toggleFavorite(contentId) {
  await api.post(`/content/${contentId}/favorite`);
  showToast('Сохранено в избранное');
}
```

### Бизнес-Правила

1. **Доступ к Контенту**
   - Весь контент доступен авторизованным пользователям
   - Некоторый премиум-контент может требовать подписку (Phase 2)

2. **Персонализация**
   - Куратор может создавать контент специально для пользователя
   - Рекомендации основаны на целях и предпочтениях пользователя

3. **Офлайн-Режим**
   - Просмотренный контент кэшируется локально
   - Видео можно скачать для офлайн-просмотра (Phase 2)

- **→ Workout Tracker**: Из тренировки

### Поведение

#### Начать Программу
```javascript
async function startProgram(programId) {
  // Проверка: есть ли уже активная программа?
  const activeProgram = await api.get('/programs/active');

  if (activeProgram) {
    showDialog({
      title: 'Активная программа',
      message: 'У вас уже есть активная программа. Завершить её и начать новую?',
      buttons: [
        { text: 'Отмена', style: 'cancel' },
        {
          text: 'Начать новую',
          onPress: async () => {
            await api.post(`/programs/${activeProgram.id}/complete`);
            await api.post(`/programs/${programId}/start`);
            navigation.navigate('ProgramDetail', { programId });
          }
        }
      ]
    });
  } else {
    await api.post(`/programs/${programId}/start`);
    navigation.navigate('ProgramDetail', { programId });
  }
}
```

#### Добавление Комментария

```javascript
async function addComment(contentId, text) {
  // Валидация
  if (text.trim().length === 0) {
    showToast('Комментарий не может быть пустым');
    return;
  }

  if (text.length > 500) {
    showToast('Максимум 500 символов');
    return;
  }

  // Отправка
  const response = await api.post(`/content/${contentId}/comments`, {
    text: text.trim()
  });

  // Добавление в список с анимацией
  appendComment(response.data.comment);
  clearInput();

  // Аналитика
  analytics.track('comment_added', {
    content_id: contentId,
    content_type: contentType,
    comment_length: text.length
  });
}
```

#### Модерация

- **Автомодерация**: Фильтрация нецензурных слов
- **Отчеты**: Пользователи могут пожаловаться на комментарий
- **Удаление**: Автор может удалить свой комментарий

### Бизнес-Правила

1. **Лимиты**
   - Максимум 500 символов на комментарий
   - Максимум 10 комментариев от одного пользователя в час

2. **Уведомления**
   - Автор контента получает уведомление о новом комментарии
   - Пользователь получает уведомление об ответе на его комментарий

3. **Куратор**
   - Комментарии от куратора выделяются (корона 👑)
   - Куратор может закрепить комментарий

### Связи

- **→ Notifications**: Уведомления об ответах

## Технические Требования

### Производительность
- Загрузка ленты контента: <1 сек
- Воспроизведение видео: <2 сек до начала
- Поиск: <500 мс

### Кэширование
- Просмотренные статьи: кэш на 7 дней
- Видео: опциональное скачивание для офлайн-просмотра (Phase 2)
- Изображения: агрессивное кэширование с CDN

### Оптимизация
- Lazy loading изображений
- Adaptive streaming для видео
- Pagination для комментариев (по 20 шт.)

### Ключевые Метрики

- **Engagement Rate**: % пользователей, взаимодействующих с контентом
- **Read Time**: Среднее время чтения статей
- **Video Completion Rate**: % просмотренных видео до конца
- **Program Completion Rate**: % завершенных программ
- **Content Feedback**: Лайки, комментарии, сохранения
