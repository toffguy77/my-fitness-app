# Привычки и Образ Жизни

## Обзор

Раздел **Привычки** фокусируется на формировании здорового образа жизни через отслеживание и развитие полезных привычек. Это дополняет основной функционал (питание и тренировки) и помогает пользователю создать устойчивые изменения в жизни.

---

## Философия

### Принципы

1. **Маленькие Шаги**
   - Лучше маленькая привычка каждый день, чем большая цель раз в неделю
   - Фокус на процессе, а не на результате

2. **Накопительный Эффект**
   - Привычки усиливают друг друга
   - Tracking one habit makes it easier to track others

3. **Гибкость**
   - Не перфекционизм, а прогресс
   - Пропуск одного дня не означает провал

4. **Персонализация**
   - Каждый пользователь создает свой набор привычек
   - Куратор помогает выбрать приоритетные

### Быстрые Действия

```javascript
// Отметить привычку выполненной
async function markHabitComplete(habitId) {
  const now = new Date();

  await api.post(`/habits/${habitId}/complete`, {
    completed_at: now
  });

  // Проверка серии
  const streak = await api.get(`/habits/${habitId}/streak`);

  if (streak.current % 7 === 0) {
    showCelebration(`🎉 Серия ${streak.current} дней!`);
  }

  // Обновление UI
  updateHabitStatus(habitId, 'completed');

  // Аналитика
  analytics.track('habit_completed', {
    habit_id: habitId,
    habit_type: habit.type,
    streak: streak.current,
    time_of_day: now.getHours()
  });
}
```

## Технические Требования

### Push-уведомления

```javascript
// Умные напоминания
async function scheduleHabitReminder(habit) {
  const optimalTime = await mlModel.predictOptimalTime(habit, user);

  await notifications.schedule({
    title: habit.name,
    body: `Время для "${habit.name}"!`,
    time: optimalTime,
    repeat: habit.frequency === 'daily' ? 'daily' : null,
    data: {
      habitId: habit.id,
      type: 'habit_reminder'
    }
  });
}

// Адаптивные напоминания
function adjustReminderTime(habit, userBehavior) {
  // Если пользователь обычно выполняет в 09:30, а напоминание в 09:00
  // → сдвигаем на 09:20
  const avgCompletionTime = userBehavior.avgCompletionTime;
  const reminderTime = habit.reminderTime;

  if (Math.abs(avgCompletionTime - reminderTime) > 15) {
    const newTime = avgCompletionTime - 10; // 10 мин до обычного времени
    updateHabitReminder(habit.id, newTime);
  }
}
```

### Офлайн-Режим

```javascript
async function markHabitOffline(habitId) {
  // Сохранение локально
  await localStorage.set(`habit_${habitId}_${today}`, {
    completed: true,
    timestamp: Date.now(),
    synced: false
  });

  // Синхронизация при восстановлении связи
  addEventListener('online', async () => {
    const pendingCompletions = await getPendingCompletions();

    for (const completion of pendingCompletions) {
      await api.post('/habits/complete', completion);
      await markAsSynced(completion.id);
    }
  });
}
```
