# Чат с Куратором

## Обзор

**Чат с куратором** — это ключевой элемент персонализированного подхода приложения. Куратор (координатор) — это персональный фитнес-тренер и нутрициолог, который ведет пользователя к его целям через общение, советы, планы и мотивацию.

---

## Роль Куратора

### Функции Куратора

1. **Планирование**
   - Создание персонального плана питания
   - Разработка программы тренировок
   - Корректировка плана на основе прогресса

2. **Поддержка**
   - Ответы на вопросы о питании и тренировках
   - Разбор ошибок и сложностей
   - Мотивация при застое или срывах

3. **Обратная Связь**
   - Еженедельные чек-ины
   - Анализ прогресса
   - Рекомендации по улучшениям

4. **Контент**
   - Персонализированные рекомендации контента
   - Создание специального контента для пользователя
   - Ответы на запросы через AI-ассистента

---

## Структура Чата

### Основные Компоненты

1. **Chat List** — Список чатов (на Phase 2 может быть несколько чатов)
2. **Chat Screen** — Экран переписки
3. **Message Types** — Типы сообщений
4. **Quick Actions** — Быстрые действия
5. **AI Assistant** — AI-помощник куратора

## 3. AI-Помощник Куратора

### Назначение
Мгновенные ответы на типовые вопросы, когда куратор недоступен.

### Принцип Работы

```javascript
async function handleUserMessage(message) {
  // Проверка: доступен ли куратор?
  const curatorOnline = await checkCuratorStatus();

  if (curatorOnline) {
    // Отправка напрямую куратору
    await sendToCurator(message);
  } else {
    // Анализ: может ли AI ответить?
    const aiCanAnswer = await analyzeMessageForAI(message);

    if (aiCanAnswer.confidence > 0.8) {
      // AI генерирует ответ
      const aiResponse = await generateAIResponse(message, userContext);

      // Отправка с пометкой "AI Помощник"
      await sendAIMessage(aiResponse, {
        disclaimer: 'Это ответ от AI-помощника. Для персональных рекомендаций дождитесь ответа куратора.'
      });

      // Уведомление куратору для проверки
      await notifyCurator({
        type: 'ai_response_sent',
        message: message,
        aiResponse: aiResponse
      });
    } else {
      // AI не уверен → ждем куратора
      await sendSystemMessage(
        'Ваше сообщение отправлено куратору. Ответ поступит в течение 2-3 часов.'
      );
    }
  }
}
```

### Типы Вопросов для AI

**AI может ответить:**
- Общие вопросы о питании (КБЖУ, калории, макросы)
- Замена продуктов
- Техника упражнений (с видео из базы)
- Расчеты (калорий, белка и т.д.)
- Поиск информации в базе знаний

**AI перенаправляет куратору:**
- Корректировка персонального плана
- Вопросы о здоровье и противопоказаниях
- Сложные или неоднозначные ситуации
- Мотивационные разговоры

## Технические Требования

### Производительность
- Отправка сообщения: <200 мс
- Загрузка истории: <1 сек
- Доставка push: <5 сек

### Реал-тайм
```javascript
// WebSocket подключение для реал-тайм обновлений
const ws = new WebSocket('wss://api.app.com/chat');

ws.onmessage = (event) => {
  const message = JSON.parse(event.data);

  switch (message.type) {
    case 'new_message':
      appendMessage(message.data);
      playNotificationSound();
      break;

    case 'message_read':
      updateMessageStatus(message.messageId, 'read');
      break;

    case 'typing':
      showTypingIndicator(message.userId);
      break;

    case 'online_status':
      updateUserStatus(message.userId, message.status);
      break;
  }
};
```

### Офлайн-Режим
- Сообщения сохраняются локально
- Автоотправка при восстановлении связи
