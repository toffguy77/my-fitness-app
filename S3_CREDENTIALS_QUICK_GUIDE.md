# 🚀 Быстрое получение S3 ключей - 3 минуты

## 📍 Прямые ссылки

### 1️⃣ Откройте консоль IAM
👉 https://console.yandex.cloud/folders/b1g7q85lgictgf4j1dq8/iam

### 2️⃣ Найдите сервисный аккаунт
Ищите: **`ajetieia8uunpq733f9t`**

### 3️⃣ Создайте ключ
1. Нажмите **"Создать новый ключ"** (вверху справа)
2. Выберите **"Создать статический ключ доступа"**
3. Нажмите **"Создать"**

### 4️⃣ Сохраните ключи

Вы увидите окно с двумя значениями:

```
┌─────────────────────────────────────────────────┐
│ Идентификатор ключа (Access Key ID):            │
│ YCAJEabcdefgh1234567                            │
│                                                 │
│ Секретный ключ (Secret Access Key):             │
│ YCMabcdefghijklmnopqrstuvwxyz1234567890AB      │
└─────────────────────────────────────────────────┘
```

⚠️ **ВАЖНО**: Скопируйте ОБА значения СЕЙЧАС! После закрытия окна секретный ключ больше не будет доступен.

## 📝 Добавьте в проект

### Откройте файл
```bash
nano apps/api/.env
```

### Добавьте в конец файла
```bash
# Yandex Cloud S3
S3_ACCESS_KEY_ID=YCAJEabcdefgh1234567
S3_SECRET_ACCESS_KEY=YCMabcdefghijklmnopqrstuvwxyz1234567890AB
S3_BUCKET=weekly-progress-photos
S3_REGION=ru-central1
S3_ENDPOINT=https://storage.yandexcloud.net
```

### Сохраните
```bash
Ctrl+O, Enter, Ctrl+X
```

## 🆘 Проблемы?

### Не можете найти сервисный аккаунт?
👉 Проверьте, что выбран правильный каталог (folder) в консоли

### Нет кнопки "Создать новый ключ"?
👉 Проверьте права доступа к сервисному аккаунту

### Забыли сохранить секретный ключ?
👉 Удалите созданный ключ и создайте новый

### Ошибка при загрузке?
👉 Проверьте, что скопировали ключи полностью (без пробелов)
